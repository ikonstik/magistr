# Система управления библиотекой

Java-приложение для управления библиотекой книг с использованием JDBC и MySQL. Реализует полный набор CRUD операций и функциональность выдачи/возврата книг.

## Структура проекта

```
src/main/java/com/example/
├── Main.java                    # Главный класс с консольным интерфейсом
├── model/
│   ├── Book.java               # Модель книги
│   ├── Reader.java             # Модель читателя
│   └── BookLoan.java           # Модель выдачи книги
├── service/
│   └── LibraryService.java     # Сервис с SQL операциями и бизнес-логикой
└── config/
    └── DatabaseConfig.java     # Конфигурация подключения к БД

sql/
├── create_database.sql         # Скрипт создания БД и таблиц
└── sample_data.sql            # Тестовые данные
```

## База данных

### Таблицы:

1. **books** (книги):
   - `id` (INT, PRIMARY KEY, AUTO_INCREMENT)
   - `title` (VARCHAR(100), NOT NULL) - название книги
   - `author` (VARCHAR(100), NOT NULL) - автор
   - `year` (INT) - год издания
   - `is_available` (BOOLEAN) - доступна ли для выдачи

2. **readers** (читатели):
   - `id` (INT, PRIMARY KEY, AUTO_INCREMENT)
   - `first_name` (VARCHAR(50), NOT NULL) - имя
   - `last_name` (VARCHAR(50), NOT NULL) - фамилия
   - `email` (VARCHAR(100), UNIQUE) - email

3. **book_loans** (выдачи книг):
   - `id` (INT, PRIMARY KEY, AUTO_INCREMENT)
   - `book_id` (INT, FOREIGN KEY) - ID книги
   - `reader_id` (INT, FOREIGN KEY) - ID читателя
   - `loan_date` (DATE) - дата выдачи
   - `return_date` (DATE) - дата возврата (может быть NULL)

## Функциональность

### CRUD операции:
-  **CREATE**: Добавление новых книг и читателей
-  **READ**: Поиск и просмотр всех записей с фильтрацией
-  **UPDATE**: Обновление информации о книгах и читателях
-  **DELETE**: Удаление записей с проверками

### Дополнительные функции:
-  Выдача книг читателям
-  Возврат книг
-  Просмотр выданных книг с информацией о читателях
-  Статистика библиотеки
-  Консольный интерфейс с удобным меню

## Требования

- Java 11 или выше
- Maven 3.6 или выше
- MySQL 8.0 или выше

## Настройка

### 1. Установка MySQL

Убедитесь, что MySQL установлен и запущен.

### 2. Создание базы данных

Выполните SQL скрипт для создания базы данных и таблиц:

```bash
mysql -u root -p < sql/create_database.sql
```

### 3. Добавление тестовых данных (опционально)

```bash
mysql -u root -p library_db < sql/sample_data.sql
```

### 4. Настройка подключения

Отредактируйте файл `src/main/java/com/example/config/DatabaseConfig.java`:

```java
config.setUsername("root");
config.setPassword("ваш_пароль"); // Замените на ваш пароль MySQL
```

### 5. Сборка проекта

```bash
mvn clean compile
```

## Запуск

```bash
mvn exec:java -Dexec.mainClass="com.example.Main"
```

## Консольный интерфейс

### Главное меню:
1. **Управление книгами** - добавление, поиск, обновление, удаление книг
2. **Управление читателями** - добавление, поиск, обновление, удаление читателей
3. **Выдача книг** - выдача книги читателю
4. **Возврат книг** - возврат выданной книги
5. **Просмотр выданных книг** - список всех выданных книг с деталями
6. **Статистика библиотеки** - общая статистика по библиотеке

### Особенности интерфейса:
- Интуитивно понятное меню
- Подробные сообщения об ошибках
- Подтверждение критических операций
- Форматированный вывод данных

## Примеры использования

### Работа с книгами:
```java
LibraryService service = new LibraryService();

// Добавление книги
Book created = service.addBook("Война и мир", "Лев Толстой", 1869);

// Поиск книг
List<Book> books = service.searchBooksByAuthor("Толстой");
List<Book> available = service.getAvailableBooks();

// Обновление
Book book = service.getBookById(1).get();
book.setTitle("Война и мир (исправленное издание)");
service.updateBook(book);
```

### Работа с читателями:
```java
LibraryService service = new LibraryService();

// Добавление читателя
Reader created = service.addReader("Иван", "Иванов", "ivan@email.com");

// Поиск
Optional<Reader> found = service.getReaderByEmail("ivan@email.com");
List<Reader> readers = service.searchReadersByName("Иван");
```

### Выдача и возврат книг:
```java
LibraryService service = new LibraryService();

// Выдача книги
boolean loaned = service.loanBook(1, 1); // книга ID=1, читатель ID=1

// Возврат книги
boolean returned = service.returnBook(1); // книга ID=1

// Просмотр выданных книг
List<LibraryService.BookLoanInfo> activeLoans = service.getActiveLoansWithDetails();
```

## Технические особенности

### Архитектура:
- **Service Layer** - все SQL операции и бизнес-логика в одном классе
- **JDBC DriverManager** - стандартное подключение к базе данных
- **Prepared Statements** - защита от SQL-инъекций
- **Транзакции** - автоматическое управление транзакциями при выдаче/возврате

### Обработка ошибок:
- Логирование всех операций через SLF4J
- Корректная обработка исключений
- Информативные сообщения для пользователя

### Транзакции:
- Автоматическое управление транзакциями
- Корректное обновление статуса книг при выдаче/возврате
